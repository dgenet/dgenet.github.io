Quest Helper

Depuis la page des talents, un clic sur l'icone du talent deploie/cache le descriptif du talent

Depuis le livre de quetes, un clic sur le bouton revele ou cache les etapes de quetes restantes pour chacune des quetes actives.

  • Pour Firefox, 

    Version 2021.5.18.1

  • Pour Chrome, 

    Telechargez le fichier, Version 2021.5.18.1, puis allez dans "chrome://extensions/", activez le mode developeur, et cliquez-deposez le fichier telecharge dans la fenetre, l'extension devrait s'installer
